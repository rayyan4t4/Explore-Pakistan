# Pakistan Tourism Map 🇵🇰

An interactive web-based tourism map of Pakistan showcasing **beautiful, historical, and popular travel destinations** across the country. Users can explore locations directly on the map and view **AI-generated overviews** for each place.

## 🌍 Features

* Interactive Map of Pakistan
* 100+ Tourist Locations
* Clickable Map Markers
* AI-Generated Place Overview
* Search Functionality
* Category Filters (Mountains, Historical, Lakes, Cities, etc.)
* Sidebar Places List
* Smooth Zoom-to-Location
* Clean and Responsive UI
* Easily Expandable Data Structure

## 🧠 Categories Included

* Mountains & Valleys
* Historical Sites
* Lakes & Beaches
* Cities & Hill Stations
* National Parks

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* Leaflet.js
* OpenStreetMap

## 🚀 How to Run

1. Clone the repository
2. Open `index.html`
3. Start exploring Pakistan 🇵🇰

No installation required. No API keys required.

## 🎯 Project Goal

To create a **simple, offline-friendly tourism exploration platform** that highlights Pakistan’s natural beauty, cultural heritage, and historical landmarks using an interactive map and AI-powered descriptions.
